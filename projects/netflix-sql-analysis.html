Overview

Business problem

The goal of this project was to use SQL to explore a real-world entertainment dataset and extract useful insights about Netflix’s content library.

This project shows my ability to write structured queries, clean data, use aggregations, filter records, and convert raw rows into meaningful insights.

Process

SQL workflow

  • Imported the Netflix dataset into MySQL.
  • Checked missing values, duplicate records, and inconsistent fields.
  • Used filtering and grouping to analyze content by type, country, and genre.
  • Wrote SQL queries to identify trends across release years and categories.
  • Converted query outputs into clear insights for portfolio documentation.
